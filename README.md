Task-4: Full Project Implementation

This project was developed as part of my Web Development Internship at ApexPlanet. It showcases my ability to build a complete and responsive web application using HTML, CSS, and JavaScript. The goal of this task was to combine all my front-end skills into one project and create a user-friendly and functional application.

The project includes multiple features such as a personal portfolio section, a to-do list with local storage support, and a product listing page with filtering and cart functionality. Users can add and remove tasks, filter products by category, add items to the cart, and complete a simple checkout process with address input and validation.

Through this task, I improved my understanding of DOM manipulation, localStorage, and responsive design. It also helped me learn how to integrate different features into a single application and build a clean and interactive user interface.
