Task-4: Full Project Implementation

The goal of this task was to combine all my front-end skills into one project and create a user-friendly and functional application.

The project includes multiple features such as a personal portfolio section, a to-do list with local storage support, and a product listing page with filtering and cart functionality. Users can add and remove tasks, filter products by category, add items to the cart, and complete a simple checkout process with address input and validation.
